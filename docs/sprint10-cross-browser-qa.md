# Sprint 10 크로스 브라우저 QA 리포트

## 범위
- 대상 화면: `/home`, `/prepare`, `/community`, `/market`, `/life`
- 대상 브라우저: Chrome, Safari(WebKit), Firefox

## 점검 항목
- 레이아웃 깨짐/오버플로우
- 주요 CTA 버튼 동작
- 폼 입력/포커스 상태
- 하단 네비게이션 및 헤더 고정 동작

## 결과 요약
- 공통: 주요 페이지 렌더링 정상
- Safari: 일부 스크롤 영역에서 탄성 스크롤 체감 차이(기능 영향 없음)
- Firefox: 카드 그림자 렌더링이 Chrome 대비 약하게 보임(디자인 이슈, 기능 영향 없음)
